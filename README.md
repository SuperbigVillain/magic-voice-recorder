# 🎙️ 魔音录屏（开源版）
一款基于 C++ / DXGI / ImGUI 开发的录屏+变声工具，主打游戏场景下的低延迟录屏与实时音频处理。

> 本软件算法与核心架构均为独家原创，已申请国家发明专利，仿冒必究。
